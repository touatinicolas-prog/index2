🚀 ÉTAPES DE MISE EN PLACE
─────────────────────────────────────────────────────────────────

1️⃣ CRÉER LE DÉPÔT GITHUB
   • Allez sur github.com
   • Nouveau dépôt (public ou privé)
   • Notez le nom : votre-username/votre-repo

2️⃣ CRÉER LE TOKEN GITHUB
   • GitHub → Settings → Developer settings
   • Personal access tokens → Tokens (classic)
   • Generate new token (classic)
   • Cochez : ☑ repo (Full control)
   • Générez et COPIEZ le token (ghp_...)

3️⃣ CONFIGURER L'APPLICATION
   • Copiez config-example.js → config.js
   • Modifiez config.js avec :
     - owner: 'votre-username'
     - repo: 'votre-repo'
     - token: 'ghp_VotreToken'
     - branch: 'main'

4️⃣ UPLOADER SUR GITHUB
   • Uploadez TOUS les fichiers sur votre dépôt
   • ⚠️ NE PAS uploader config.js (sensible !)
   • Vérifiez que .gitignore est présent

5️⃣ ACTIVER GITHUB PAGES
   • Dépôt → Settings → Pages
   • Source : Deploy from a branch
   • Branch : main / root
   • Save

6️⃣ ACCÉDER À L'APPLICATION
   • URL : https://votre-username.github.io/votre-repo/
   • Ajoutez aux favoris sur iPad/iPhone
   • Mode plein écran : "Sur l'écran d'accueil"
